# AugieChat

Simple web chat bot powered by OpenAI with configurable branding.

## Setup

1. Install dependencies (done automatically in Codex via `.codex/setup.sh`):

```bash
npm install
```

2. Create a `.env` file or set the `OPENAI_API_KEY` environment variable with your OpenAI API key.

3. (Optional) Edit `branding.json` to customize site name, logo path, fonts and colors.

4. Start the development server:

```bash
npm start
```

Then open `http://localhost:3000` in your browser.
